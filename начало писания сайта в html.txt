начало написания сайта в Html
<!DOCTYPE HTML>
<html>
    <head>
       <meta charset="UTF-8">
       <title>ник сайта</title>
    </head>
    <body>
        текст сайта
    </body>
</html>

теги:
<br>- служит для переноса текст на другую строчку 
<p>...</p>- параграф 
        <ol>
            <li>илья</li>
        </ol>- создает перечисление текста
<img src="путь к каринке">- для всавления картинки на сайт
widht="px"- ширина
height="px"- высота
<div>...</div>- блоки 
<input>- ввод текста 
<input value="...">- текст в я чейке ввода 
<input placeholder="..."- текст в ячейке ввода который не нужно стирать
<textarea>...</teztarea>- текст для большого ввода данных
<header>...</header>- шапка сайта
<footer>...</footer>- подвал сайта
<section>...</section>- раздел сайта
<nav>...</nav>- навигация по сайту
Пунктиром выделен особый тег <body>.
Синей рамкой выделяются поточные теги, которыми обычно размечают крупные структурные блоки страниц, например <main>.
Фиолетовой рамкой выделяются теги для создания смысловых разделов, например <section>.
Оранжевой рамкой выделяются заголовочные теги, например <h1>.
Розовой рамкой выделяются поточные теги, которыми обычно размечают непосредственно текстовые элементы, например <p>.
